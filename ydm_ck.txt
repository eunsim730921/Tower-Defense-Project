프롬프트 정리

1. [초반 정보 입력 + 타워 클릭 시 맵 우측 타워 정보 표시]
나는 소프트웨어 학과 3학년이고, 기존 웹 게임에 기능을 추가하는 과제를 하고있어 
내가 올린 파일은 기존 웹 게임 코드이고 협업 방식으로 진행 중이야 
나는 타워 관련
    (타워 종류, 
     타워 사거리, 
     타워 공격력, 
     타워 공격속도, 
     타워 강화, 
     타워 팔기, 
     타워에 마우스 가져다 대면 사거리 표시, 
     타워 공격모션 수정) 
로직을 담당해.
팀원은 적과 UI를 담당하기로 했어. 
우선 타워에 마우스를 가져다 대면 id=gameCanvas 옆에 width="250", height="450"의 UI가 나타나게 만들어줘

[피드백]
지금 타워 클릭시 UI가 생기면서 gameCanvas가 고정되지 않고 움직여. gameCanvas의 위치는 고정시키고 그 옆에 타워 UI가 나타나게 해줘

2. [타워에 마우스 올리거나 클릭 시 사거리 표시]
타워를 클릭하거나 마우스를 올려둘 때(hover) 사거리가 표시되게 해줘. 클릭을 했을 때는 다시 클릭을 하기 전까지 사거리 표시가 유지돼야해

3. [맵 격자 형태로 변경]
맵 형태를 격자형태로 만들어서 모든 부분에 타워를 설치할 수 있게 해줘. 
격자 형태는 시각적으로 볼 수 있어야하고, 적 이동경로에는 타워를 설치할 수 없어

4. [tower_defense.js , tower.js 기능별 별도 js 파일 분리]
tower_defense.js와 tower.js를 기능별로 별도의 js 파일로 분리해줘. 
각 파일은 한 기능에만 집중하도록 만들고, import/export 없이 전역 변수로 상호작용하는 구조로 만들어줘. 
대신 tower_defense.js와 tower.js의 기능이 파일을 변경했음에도 변함없이 똑같이 작동하고 
file:// 환경에서도 바로 실행 가능해야 해 . 
    ex) main.js / map.js / enemy.js 등등

5. [타워 종류 추가]
Have a Break, and then.. 먼저 이 질문에 답하기 위한 최적의 접근 전략을 설계하고, 
그 전략에 따라 체계적으로 문제를 해결해줘 
[질문] 현재 격자 형태로 된 맵의 타일을 클릭하면 바로 타워가 생성 돼. 
이 타일을 누르면 타일이 연한 파란색(선택된 타일이라는 뜻)으로 바뀌고,
id="towerInfoPanel" 여기에 타워 종류가 뜨고 원하는 타워 종류를 선택하면 해당 타워를 타일에 배치하게 만들어줘

6. [타워팔기 기능 추가]
타워를 판매할 수 있는 기능을 추가해줘. 
각 타워가 건설 및 강화에 사용한 모든 금액을 하나의 변수(investedMoney)에 저장하도록 하고, 
팔기 버튼을 누르면 해당 타워의 투자 금액의 70%를 환급받도록 만들어줘. 
기존 tower.js의 다른 기능(타워 생성, 강화, 공격, 사거리 표시 등)은 변경 없이 그대로 유지되어야 해. 

7. [스플래쉬, 슬로우 타워 변경]
스플래쉬 기능을 슬로우 타워가 사용하도록 하고 스플래쉬 타워는 화염 타워로 교체해줘
화염 타워는 단일 공격이지만, 공격을 하고 2초간 지속적으로 적에게 피해를 주는 burn 특성도 추가해줘
슬로우와 화염이 적용된 적은 그에 맞는 상태이상을 표현해줘.

8. [투사체 이펙트 추가]
타워 공격 시 실제로 투사체가 날아가 적에게 맞으면 피해를 주는 시스템을 만들어줘.
각 타워는 고유한 색상, 속도, 크기의 투사체를 가져야 하고, 종류에 따라 시각 효과도 달라야 해.
지금처럼 검은색 선으로 공격선을 표시하는 대신, 투사체가 직접 화면에 보이게 해줘.
화염 타워와 슬로우 타워는 특별한 공격 방식과 시각 효과를 가지도록 개선해야 한다.
기존의 타워 배치, 강화, 판매, 사거리 표시 같은 다른 기능은 변경하지 말고 그대로 유지해줘.

9. [인트로, 아웃트로 추가]
게임 시작 전 인트로 화면과 게임 종료 후 아웃트로 화면을 추가해줘.
인트로에는 “게임 시작” 버튼만 보이게 하고, 게임 시작 전엔 체력·돈·웨이브 UI가 숨겨져 있어야 해.
게임오버 시 전체 화면에 “게임 오버”와 “다시 시작” 버튼을 표시하고,
“다시 시작”을 누르면 UI와 타워가 초기화된 뒤 인트로로 돌아가도록 만들어줘.

==============================

개선한 점

1. 고정된 타워 위치를 맵을 격자 형태로 만들어 적 이동경로를 제외한 모든곳에 타워 배치 가능
2. 게임 화면 우측에 타워 선택, 정보 UI 추가
3. 기존 한가지 종류의 타워를 기본, 속공, 강타, 슬로우, 화염 5가지 종류로 변경
4. 타워 공격력, 공격 속도, 사거리 강화 기능 추가
5. 타워에 사용한 금액(타워 생성, 강화)의 70%를 돌려받는 팔기 기능 추가
6. 인트로, 아웃트로 화면 추가
7. 기존 검은색 선으로만 보였던 타워 투사체를 타워 각각 다른 색의 원형 투사체로 변경(슬로우 타워는 파동형태로 광역 슬로우)
8. 타워에 마우스를 올려두거나 클릭을 했을 경우 타워 사거리 표시
9. 하나의 js 파일에 모든 기능을 넣어 코드 가독성과 유지보수가 떨어져 기능별로 별도의 파일로 분리
